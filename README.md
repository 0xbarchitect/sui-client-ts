# sui-client-ts
Typescript client code to interact with SUI Defi ecosystem.

## Prerequisites

- [NodeJS v20](https://nodejs.org/en/blog/release/v20.9.0)

## Setup

- Create `.env` file from template and fill in secrets

## Install 

```sh
$ npm install
```

## Build 

```sh
$ npm run build
```

## Run

```sh
$ npm run start
```
